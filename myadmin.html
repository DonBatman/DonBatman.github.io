myadmin

myadmin is a mod to add admin picks and chat commands and privs and other server related things.

There are 2 admin picks. One just destroys and the other destroys but adds the items to your inventory.

mybadnames will refuse a player from joining if their name contains a bad word.

mybedrock sets an unbreakable layer of stone at a certain level. It is set at 29000 deep right now but can be changed in settings.

The chat commands adds emotions to the chat. For example /happy will put playername is happy! in the chat. /emotes gives you a list of chats available.

mycursing kicks people from the server if they curse in the chat. It does not ban them and they can rejoin the server.

myextras is for little projects I make. Right now it only has one. It sets the hotbar to different widths. You can set it from 1 to 16 wide. You use a chat command to change it. Example - /setbar 12

myfirstthings gives a player items when they first join the server. You can change what they get in init.lua

myitempickup picks up items near you. It is helpful for picking up things that mobs drop or saplings when you cut down a tree.

myprivs is an easy way to give provs to players. There are different levels for different players. From basic player all the way to super admin.

myspawn gives you different spawn locations. You just need to set it in init.lua. Edit the file to set spawn locations. It is set up with 3 right now. You can easily copy, paste and edit commands to get more.

mystart is a page that pops up when a player first joins the game. It can have rules or info about the server and more. You just need to edit the rules.txt file.

myunderworld is a teleportation device. Set the teleporter down, give it a name and set the depth. It sets a teleporter at that depth. Great for setting up mines.
