myconcrete

Myconcrete adds concrete and concrete items to Luanti. It adds limestone for you to mine so you can make concrete.

  • Minetest Forum Link - myconcrete
  • Github Link - Veiw Code
  • Direct Download - myconcrete.zip
  • ContentDB - myconcrete



To get started you need to mine limestone. When you mine it you get lime.


If you want limestone you need to cook the lime.


Once you have lime you will need gravel. Mix the 2 together in the crafting table to get concrete.


Here are the recipies for the items.
