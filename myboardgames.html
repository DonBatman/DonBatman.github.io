myboardgames


myboardgames adds some mini games to Luanti. Place the block where you want the game. The game will set up North and East of that spot. I have included the sizes of the games. (Pitfall is long so be careful) It will remove anything that is in its way so be careful to place it right. Once placed just right click on the block and it will load the game.
Have fun playing the game! When the game is over just right click on the placer block and the game resets.
There are no crafts so someone with the right privs will have to get the placer node and place the game.

  • Checkers - Schematic size(xyz) - 12 x 3 x 12

  • Canadian Checkers - Schematic size(xyz) - 16 x 3 x 16

  • Draughts - Schematic size(xyz) - 14 x 3 x 14
